### Hinzugefügt
- Initial Release: ICAP Test System
- Python ICAP-Server (~200 Zeilen) mit ClamAV-Integration
- ICAP Test-Client mit EICAR und Clean File Tests
- Docker Compose Setup (ClamAV + ICAP-Server)
- Vollständige Dokumentation mit Mermaid-Diagrammen
- Unterstützung für ICAP/1.0 (OPTIONS, REQMOD, RESPMOD)
- Direkte ClamAV-Integration via TCP INSTREAM
- Alpine Linux basiertes Docker-Image (~50 MB)
